Getting Started
After purchase, our engineer will DM you on Discord to set up your node. Share your preferred server version and any plugins you need.
Pelican Panel Access
Log in at panel.zqhost.qzz.io using the credentials provided by our team. From there you can manage files, view console, start/stop your server, and install mods.
Backups
Managed backups are created automatically based on your plan. You can restore any backup directly from the Pelican Panel with one click.
Plugin Installation
Upload .jar plugin files to the /plugins folder via Pelican Panel's file manager and restart your server. Gold & Diamond plans include a curated premium plugin pack.
Support
For any issues, open a ticket in our Discord server. Our team responds typically within a few hours.
